Advantages pf the client/server model

Several major advantages of the client server are:

  1. Divides Application Processing: across multiple machines.Non-critical data and functions are processed on the client Critical functions are processed on the server
  2. Optimizes Client Workstations for data input and presentation (e.g., graphics and mouse support)
  3. Optimizes the Server for data processing and storage (e.g., large amount of memory and disk space)
  4. Reduces Data Replication - Data stored on the servers instead of each client, reducing the amount of data replication for the application.

Example of TCP/UI Protocol

  • FTP-File transfer protocol;allows files to be transfered between computers.
  • HTTP-Hyper text transfer protocol;it is a network protocol that allows you to access web page links.
  • HTTPS-Hyper text transfer protocol secure; it is an extension of HTTP it is a secure version of it.
  • DHCP-Dynamic Host Configuration Protocol; is a protocol that automatacially assigns an Ip address to a device that connects to a network.
  • SMTP-Simple Mail Transfer Protocol; a communication protocol for electronic mail transmission.

Examples of network utilities and services

  • ping- allows you to test the availiablity of a host
  • traceart-a computer network diagnostic tool for displaying the route and measuring transit delays of packets across an Internet Protocol network
  • pathping-Windows based command-line tool used to provide information about the path data takes to its intended destination, network latency and network loss
  • whois-looks up information about domain names
  • ipconfig-a program that displays TCP/IP network Configuration and domain name settings
